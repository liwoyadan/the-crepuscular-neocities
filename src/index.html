The Crepuscular

Welcome to The Crepuscular, an unofficial fansite for the closed species ARPG Stygian-Shade. This site exists to house some various guides to the Stygian ARPG, some helpful user tools, character profile codes, event archives, and more...! It also serves as a website to hold some of my own personal Stygian profiles and worldbuilding.

This site is heavily under construction and is only being worked on as a fun side project, so please excuse the mess. If you happen to come across any bugs (including anything breaking on mobile) or inconsistencies, let me know in the guestbook and I'll check it out!
